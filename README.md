# Absent Roll Number Finder

## Excel format

### Master file

Columns:

- ID or UniqueId or Student UniqueId
- Name
- Roll No or HTNO

### Absent file

One column:

- ID or UniqueId

## Run in StackBlitz

1. Open StackBlitz.
2. Create a React + Vite project.
3. Upload all files from this folder.
4. StackBlitz installs the packages automatically.
5. Upload the master Excel file.
6. Upload the absent-ID Excel file.
7. The page displays only matched roll numbers.
